Это приложение риелтор.  Написано на языке Kotlin для телефонов на Android.

Оно хранить информацию о квартирах и домах, ищет нужные варианты по разным запросам и добавлять просмотр квартиры в календарь телефона.

Всё сделано в программе Android Studio. База данных хранит список домов, квартир их площадь, тип санузлов.
